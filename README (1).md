# 連連看遊戲 (index.html)

## 檔案功能分析

### 遊戲玩法
- **棋盤**: 8x12 (96張卡片，48對)
- **目標**: 配對相同圖標的卡片來消除
- **路徑規則**: 支援直線、一轉彎、兩轉彎三種連接方式
- **時間限制**: 180秒

### 功能列表
| 功能 | 說明 |
|------|------|
| 計時器 | 倒數計時顯示 |
| 計分 | 配對成功 +10 分 |
| 💡 提示 | 顯示可配對的卡片 (冷卻10秒) |
| 🔀 洗牌 | 隨機重新排列未配對卡片 (-50分) |
| 💥 消除 | 自動消除3對 (-100分) |
| 🔄 重新開始 | 重置遊戲 |

### 視覺效果
- 深色主題 (#1a1a2e 背景)
- SVG 動態路徑繪製
- 卡片動畫 (選取、消除動畫)
- 提示動畫 (閃爍效果)
- RWD 支援 (手機版)
